# IT Handoff Documentation - English

This folder is the 2026-06-08 module-based specification for near-term IT implementation.

## Scope

This handoff covers these roles and touchpoints:

- `Requester`
- `Cost Manager` (legacy label in some files: `Manager B`)
- `Dept DRI`
- `Budget Approver`
- `OM Purchasing`
- `Buyer Handoff`

MFG Coordinator, Sourcing, and Buyer appear only as downstream touchpoints and are not fully specified in this package.
