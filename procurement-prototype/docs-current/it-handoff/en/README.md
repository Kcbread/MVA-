# IT Handoff Documentation - English

This folder is the live module-based specification for near-term IT implementation. It was reviewed against the current repo context on 2026-06-15.

## Scope

This handoff covers these roles and touchpoints:

- `Requester`
- `Cost Manager`
- `Dept DRI`
- `Budget Approver`
- `OM Purchasing`
- `Buyer Handoff`

MFG Coordinator, Sourcing, and Buyer appear only as Buyer Handoff touchpoints and are not fully specified in this package.
