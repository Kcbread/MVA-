目前系統流程

User A / DRI
需求提出

透過 Request、History、Create New Item 建立需求，Demand Tracking 僅作檢視。

Manager B
審批決策

直接 Approve / Return / Reject，並透過 Project Setup 管理項目開放狀態。

OM Purchasing
IT / OM Buy

負責 PAS 資訊、需求彙整、報價資料、Quote PDF 與 OM Excel package。

MFG Coordinator
資料包收斂

檢查 line / phase 資訊是否收齊，整理 MFG package 與 RFQ 輸出。

Buyer / 外部系統
PR / PO 追蹤

記錄外部系統結果、PR No.、PO No.、退件原因與佐證附件。

OM Excel Package 正式輸出給外部系統的 OM 採購資料包。
Quote PDF 報價單佐證，與 Excel package 一起追蹤。
MFG RFQ Excel MFG 詢價 / 資料包輸出格式模擬。
Evidence Timeline 截圖、貼上結果、email、PDF、Excel、zip 佐證。
