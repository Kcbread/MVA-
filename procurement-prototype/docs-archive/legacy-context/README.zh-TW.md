# Legacy Context Archive

此資料夾保存過去產出的舊流程圖、舊角色、舊 Kuso/IT 視覺化相關文件。這些文件可能仍有參考價值，但不再作為新 thread、subagent、PM handoff 的預設上下文來源。

## 使用規則

- 日常開發請先讀 `procurement-prototype/_context/README.zh-TW.md`。
- 只有在使用者明確要求追溯舊流程圖、舊 Kuso brief、舊 IT 文件時才讀本資料夾。
- 若本資料夾內容與 `_context/` 或 `PROJECT_DECISIONS.md` 衝突，以 `_context/` 與 `PROJECT_DECISIONS.md` 為準。
