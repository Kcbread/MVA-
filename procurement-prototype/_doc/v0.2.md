# v0.2 - Manager / OM Consolidation + Test Contract

## Completed Changes
- Manager B top-level tabs are consolidated to `Approval`, `Demand Analysis`, `Progress Tracking`, and `Project Setup`.
- `Approval` now contains both `Pending Approval` and `Approval History`.
- `Demand Analysis` keeps a first-layer `Cost Dashboard` and second-layer `Station Matrix`.
- Cost Dashboard is changed to an Excel Dashboard-style `Item x Unit` table: item/spec/price plus MFG and Non-MFG unit columns.
- OM Purchasing tabs are consolidated to `Submission Dashboard`, `PAS Demand No`, `PAS Quote Result`, `Quote Expiry`, and `Export Package`.
- `PAS Quote Result` now contains quote validity fields inside the `Quote Result` card.
- `Quote Expiry` is a separate overview tab for validity tracking and 14-day expiry warnings.
- Temporary Budget UI is scoped to OPM New Request; Manager / OM / Admin table-wide injection is disabled.

## Module Work
- Added `app-modules/quote-validity.js` for quote validity and currency display helpers.
- Added `app-modules/demand-cost-dashboard.js` for item x unit dashboard aggregation.
- `app.js` remains oversized and must continue to be reduced in future versions; v0.2 begins the split with testable business helpers.

## Tests
- Added `test.sh` as the single project test entrypoint.
- Added Node unit tests for quote validity, currency conversion, and demand cost aggregation.
- Added system contract tests for Manager tabs, OM tabs, PAS Quote Result table structure, and removal of active MutationObserver injection.

## UI Risk Notes
- Browser smoke depends on local Playwright browser availability. If unavailable, `test.sh` reports a skip instead of pretending the browser test passed.
- Future UI changes must update `tests/system-contract.test.js` and this `_doc/vx.y.md` file in the same change.
