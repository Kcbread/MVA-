# v0.5 - Manager Phase x Unit Dashboard + OM Workflow Simplification

## Summary
- Manager B `Demand Analysis` now uses a first-layer `Cost Dashboard` modeled after the Excel Dashboard sheet: `Item x Phase x Unit`.
- `Station Matrix` remains the second-layer drilldown for station-level reasonableness checks.
- OM Purchasing remains a four-step workflow: `Submission Dashboard / PAS Demand No / PAS Quote Result / Export Package`.
- `Quote Expiry` is not a workflow tab. Quote validity is entered in `PAS Quote Result` and monitored from `Submission Dashboard`.

## Manager B Decisions
- Summary cards now focus on business signals:
  - Total Estimated Amount
  - MFG Amount
  - Non-MFG Amount
  - Highest Unit
  - Highest Phase
  - Price Pending
- Cost Dashboard rows are item/spec/price, then phase groups from `P1.0` to `MP`.
- Each phase group has unit columns:
  - MFG
  - FATP TE
  - FATP IQC
  - FATP PQE
  - WH
  - Q-LAB
  - REL
  - ENG1
  - ENG2
  - ENG3
  - IT
  - FAC
- Clicking a populated cell opens `Station Matrix` with item, phase, and unit filters applied.
- Demo data now includes both MFG station rows and Non-MFG unit rows so the dashboard can be reviewed with realistic populated data.

## OM Purchasing Decisions
- `Submission Dashboard` tracks OM operational aging:
  - Received Date
  - Current Stage
  - Days in Stage
  - Next Action
  - Quote Expiry
  - Pending Reason
- Removed Budget/PR/PO/Arrived-heavy columns from the OM dashboard main table.
- `PAS Demand No` only records the PAS Demand No.
- `PAS Quote Result` is the only place to enter:
  - PAS Material No
  - Vendor
  - Vendor Part No.
  - Unit Price
  - Quote Date
  - Quote Received Date
  - Quote Valid Until
  - Quote PDF
  - Quote Excel
- `Quote Valid Until` remains required before sending to User A.

## Cache / Regression Guard
- OM tab state now falls back to `Submission Dashboard` if a stale tab name such as `quoteExpiry` is encountered.
- Script query version was bumped to force local file reload.
- System contract tests assert there is no top-level `Quote Expiry` tab or panel.

## Tests
- `tests/system-contract.test.js` covers:
  - Manager `Demand Analysis` has `Cost Dashboard / Station Matrix`.
  - Manager Cost Dashboard is dynamically rendered as phase x unit.
  - OM tabs remain the four official workflow tabs.
  - OM Submission Dashboard uses `Days in Stage` and `Next Action`.
- `tests/unit.test.js` covers:
  - Quote validity 14-day threshold.
  - Currency conversion.
  - Demand Cost Dashboard item/unit aggregation.
  - Phase/unit aggregation.
